# Hotel-Management-System

**Background**

The Hotel Management System is a Java-based desktop application developed to streamline daily hotel operations. The system supports efficient room booking, guest management, and billing processes through a structured, role-based interface. It allows hotel staff to manage reservations, check-ins, check-outs, employee records, and financial transactions, while providing guests with a smooth room reservation experience. By organizing operations digitally, the system reduces manual errors, improves accuracy, and enhances overall service quality.

Originally developed in (2024-2025 / 2nd Term) as part of an academic course, and uploaded to GitHub later for documentation and portfolio purposes.

**System Features**

- Role-based access control (Admin, Receptionist, Guest)

- Room booking and reservation management

- Check-in and check-out processing

- Employee management

- Payment and billing system

- Report generation for rooms and employees
